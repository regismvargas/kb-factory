# KB Factory documentation

Not sure where to start? Pick the path that matches you.

## 🟢 I'm not a developer — I just want to use it by chatting

Start with the **[No-code guide](guide/no-code/index.md)**. It's a gentle,
zero-command walkthrough: what KB Factory is in plain words, a hand-held first
session, and a plain-language dictionary for any term you meet.

- [What this is](guide/no-code/what-this-is.md) — the idea, in everyday words.
- [Your first session](guide/no-code/first-session.md) — set up and record your
  first thing, just by asking.
- [The words it uses](guide/no-code/words.md) — plain-language dictionary.
- [When something's not working](guide/no-code/when-stuck.md) — quick fixes.

## 🔵 I'm comfortable in a chat (Claude Code / Cowork)

The **[User Guide](guide/index.md)** is the full, task-oriented walkthrough for
working with KB Factory in a conversation — what to say, how sessions flow, which
plugins to pick, and recipes for common situations.

- [Install and first session](guide/install-and-first-session.md)
- [Everyday use](guide/everyday-use.md) · [How it works](guide/how-it-works.md)
- [Plugins](guide/plugins.md) · [Recipes](guide/recipes.md) ·
  [Troubleshooting](guide/troubleshooting.md)
