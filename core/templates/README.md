# Templates

Canonical KB scaffold templates.

## kb/

The `kb/` directory contains the complete scaffold that every new project receives. It is the validated scaffold that `kb.py init` installs into a new project.

Contents:
- `kb.py` — stdlib-only CLI runtime (~850 lines)
- `kb.config.json` — project configuration with template placeholders
- `SKILL.md` — agent protocol for operating the KB
- `memory/` — warm start files (NOW.md, HOT.md, INDEX.md, topics/)
- `exports/` — platform export directories (cowork/, claude-ai/)
- `seed/` — initial records template (JSONL with placeholders)

## Rules

- `kb.db` is never included in the template. It is generated by `kb.py init`.
- Template placeholders use `{{VARIABLE}}` syntax.
- This snapshot is immutable until explicitly updated by a future release.
